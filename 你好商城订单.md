### 订单在各个状态下的可操作行为

- 待付款 

> * 取消订单(买家)
> * 付款(买家)

- 待确认

> * 确认(**卖家**)
> * 取消订单(买家)

- 待发货

> * 修改地址(买家)
> * 退款(买家)
> * 批量退款(买家)
> * 确认发货(**卖家**)

- 待收货

> * 确认收货(买家)
> * 退换(买家)
> * 批量退换(买家)
> * 查看物流(买家)

- 交易成功

> * 评价(买家)
> * 删除订单(买家)
> * 退款(买家)
> * 退货退款(买家)
> * 查看物流(买家)

- 已取消

> * 删除订单(买家)
